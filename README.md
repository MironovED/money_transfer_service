## Курсовой проект на тему "Сервис по переводу денег"

Курсовой проект представляет собой серверную часть приложения по переводу денег с одной карты на другую

### Запуск проекта: 

Сначала необходимо собрать докер образ:
`docker build -t app:latest -t .`

Запаковать приложение в архив: `mvn clean package`

Запустить приложение в докере: `docker-compose up`
